This module import nodes from any site to drupal site.


Installation
-------------

* Copy the whole "node_generator" directory to your modules
  directory - "sites/all/modules"
* Enable the module.
* Set your desired bundle wise permissions at - "admin/people/permissions".


Usage
------
   
 * Go to "/admin/config/system/metatags-upload/node-generator-form".
 * Fill required fields in form.

Permissions available
---------------------
* Access Node Generator Form
